Reestar Restaurant Webpage

Reestar is a simple restaurant webpage built using **HTML and CSS**.  
This project displays a modern food menu layout with a clean and user-friendly design.

Live Demo
https://harish-20cs022.github.io/Reestar-webpage/

Features
- Simple and clean UI
- Food menu card layout
- Responsive design
- Contact section
- Built using pure HTML & CSS

 Technologies Used
- HTML5
- CSS3

This webpage contains:
1. Navigation bar
2. Food menu section
3. Contact form
